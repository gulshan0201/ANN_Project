# ANN_Project Customer churn prediction

## Project Overview

This project demonstrates a step-by-step implementation of a machine learning workflow using a Jupyter Notebook. It covers the complete pipeline from data preprocessing to model building, training, and evaluation.

## Key Features
Data loading and exploration
Data preprocessing and cleaning
Feature selection and transformation
Model building using machine learning algorithms
Training and evaluation of the model
Performance analysis with appropriate metrics

## Technologies Used
Python
Jupyter Notebook
NumPy
Pandas
Scikit-learn
Matplotlib / Seaborn (for visualization)

## Project Workflow
Import required libraries
Load and inspect dataset
Handle missing values and preprocess data
Perform exploratory data analysis (EDA)
Split data into training and testing sets
Train machine learning model
Evaluate model performance
Visualize results
